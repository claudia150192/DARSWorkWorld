Question - In-Kind Support With Value of One-Third Reduction (VTR)

Select “Yes” if your in-kind support from others in your household is considered “Full” (that is, it is figured using the Value of the One-Third Reduction [VTR]), or “No” if it is not.

Additional Information

More information about In-Kind Support With Value of One-Third Reduction (VTR) and how it is calculated is available from the WorkWORLD Help/Information System online at:
