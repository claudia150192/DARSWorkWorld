Question - Benefits Planning Query [BPQY]

Select “Yes” if you have a current Benefits Planning Query (BPQY) report from the Social Security Administration (SSA), or “No” if you do not.

The Benefits Planning Query (BPQY) provides a snapshot of your benefits and work history as it is stored in the Social Security Administration's (SSA's) electronic records. You may request this free report as often as you like, and use it to help ensure that information SSA has about you is current and correct.

The BPQY has much of the information you need to complete WorkWORLD's current situation. It is designed to communicate the essential facts as recorded by SSA, and shows in one place information that SSA stores about you in a number of databases.

Additional Information

Complete information about the BPQY report and how to obtain it is available from the WorkWORLD Help/Information System online at:
